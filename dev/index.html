bfabricPy Documentation#

bfabricPy provides a Python interface to the B-Fabric system.

This documentation is organized primarily by what you want to do rather than by API structure:

  • Getting Started - Installation, configuration, first script

  • User Guides - Task-based tutorials and examples

  • API Reference - Complete, auto-generated documentation

  • Advanced Topics - Advanced features and patterns

  • Design Notes - Architecture and design rationale (for contributors)

  • Resources - Error handling, best practices, contributing
